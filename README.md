# Delivery App - Fullstack Project

This is a comprehensive food delivery application featuring a React-based frontend and a Node.js/MongoDB backend.

## 🚀 Public URL

**Frontend:** [(https://delivery-app-ten-roan.vercel.app/)]  
**Backend API:** [https://delivery-app-bhuj.onrender.com/]

## ✅ Accomplished Tasks (Level: Advanced)

- [x] **Base Level:** Core shopping functionality, cart management, and order placement.
- [x] **Database Integration:** Real-time data fetching from MongoDB Atlas (No JSON files used).
- [x] **Responsive Design:** Fully functional on Desktop, Tablet, and Mobile.
- [x] **Product Filtering:** Filter items by categories (Burgers, Drinks, Desserts).
- [x] **Product Sorting:** Dynamic sorting by price (Asc/Desc) and Name (A-Z).
- [x] **Shop Rating System:** Filtering restaurants based on their user ratings (e.g., 4.0+ stars).
- [x] **Form Validation:** Client-side validation for the checkout process to ensure data integrity.

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Styled Components.
- **Backend:** Node.js, Express, Mongoose.
- **Database:** MongoDB Atlas (Cloud Instance).
